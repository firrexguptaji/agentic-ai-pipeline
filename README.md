# Agentic AI Pipeline

## Overview

This project implements a modular agentic AI system with:

* LLM-based agent
* Retrieval-Augmented Generation (RAG)
* Tool calling (planned)
* Microservice architecture

## Current Scope (MVP)

* User query → Agent → RAG → Response

## Architecture (Planned)

* API Gateway
* Agent Service
* RAG Service
* Embedding Service
* Vector Database
